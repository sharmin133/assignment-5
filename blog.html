Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is get or sets the HTML content inside an element, including tags.innerText is gets or sets the visible text inside an element,ignoring hidden elements and extra spaces. textContent gets pr sets all text inside an element, including hidden elements,but without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single parent elemnet handles events for multiple child elements even if they added dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates upward through its parent elements until it reaches the document root.
